   • Toque no botão 🎤 (fica verde quando ativo)
   • Permita acesso quando solicitado
   • Aguarde aparecer "Escutando..."

   • Aguarde o sistema processar
   • Se não funcionar, use o chat por texto
   • Botão 🔧 para testar conexão
